# FileCryptor

The simple file encryptor and decryptor

based on logical XOR operation

## Dependencies
* cmake
* C++ complier (gcc or clang)
* Boost library

## Installation
1. `mkdir build`
1. `cd build`
1. `cmake .. && make`
1. Run `./fcryptor -h` for show help
