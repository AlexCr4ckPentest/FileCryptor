# FileCryptor

The simple file encryptor and decryptor

based on logical XOR operation

## Dependencies
* CMake (>= 3.1)
* C++ complier (with C++17 support)
* Boost library (>= 1.67)

## Installation
1. `mkdir build`
1. `cd build`
1. `cmake .. && make`
1. Run `./fcryptor -h` for show help
